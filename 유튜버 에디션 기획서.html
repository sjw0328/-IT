01 · 개요

왜 유튜버 에디션인가?

광고비 없이 팬덤에게 도달하는 가장 현실적인 방법. 유튜버에게는 자신의 이름을 건 앱 테마를, 팬들에게는 유튜버를 연상시키는 앱 경험을 제공한다.

💸
무비용 노출
유튜버가 자기 채널에서 직접 소개할 동기를 만든다. 광고 집행 없이 수만 뷰에 노출 가능.
🎯
타깃 정밀 도달
토론·시사 콘텐츠 시청자는 토론하자의 핵심 잠재 사용자. 팬덤 = 타깃.
🔁
확장 가능 시스템
월수·세신사 에디션 → 다른 유튜버 에디션으로 반복 확장. 에디션이 쌓일수록 앱의 콘텐츠 자산이 됨.

06 · 협업 전략

월수에게 연락하는 방법

월수는 시청자 이메일·DM을 읽고 반응한 전례가 있다. 팬이 보내는 제안처럼 접근하되, 실제로 구현된 에디션을 보여주는 게 핵심이다.

1
이미 만들어 놓고 보여준다
말로 설명하지 말고 링크 하나를 준다. "만들어뒀는데 써주실 생각 있으세요?" — 이미 완성된 걸 보여주면 거절 비용이 낮아진다.
✓ 이미 구현 완료
2
팬의 언어로 쓴다
비즈니스 제안처럼 쓰면 무시당한다. "유튜브 보다가 만들었는데요" 식의 팬 메시지가 훨씬 잘 읽힌다. 짧게, 진심 있게.
3
요구 사항은 딱 하나만
커뮤니티 글 하나, 또는 쇼츠 1개에 링크 언급. 협찬·대가 요구 없이 "써봐도 재밌을 것 같아서요" 선에서 마무리.
이메일 초안 (참고용)
받는 사람: 월수 채널 이메일 / 커뮤니티 DM 제목: 월수 에디션 만들어봤어요
안녕하세요 월수님, 유튜브 오래 봐온 시청자입니다. 저희가 만든 토론하자(toronhaja.app)라는 토론 앱에 월수 에디션을 만들었어요. 앱 열고 헤더에 있는 🎮 에디션 버튼 누르면 바로 확인하실 수 있습니다. 김근육 썸네일 넣고, 색상도 바꾸고, 로고 옆에 "× 월수" 붙여놨어요. 돈 드리거나 뭔가 요구하는 건 아니고요, 재밌으시면 커뮤니티에 한 번 올려주시면 감사하겠습니다. (별로면 그냥 무시하셔도 됩니다 ㅎㅎ) 감사합니다. 토론하자 서주원 드림

08 · 콘텐츠 아이디어

에디션을 진짜 콘텐츠로 만드는 10가지

에디션이 "테마만 바뀐 것"에서 벗어나려면 안에 속살이 있어야 한다. 유튜버 팬이 에디션을 켰을 때 느끼는 차이가 명확해야 소개할 이유가 생긴다.

1
유튜버 토론 테마 팩
에디션 활성화 시 해당 유튜버 콘텐츠 주제로 구성된 전용 토론 세트가 피드 상단에 자동 표시. 월수라면 병역·사회풍자·남성 담론, 세신사라면 북한·통일·국방. 팬이 "이 에디션은 이런 주제다"를 즉시 체감.
✓ 현재 picks 섹션으로 구현됨 — 주제 확충 필요
2
유튜버 어록 배너 슬라이드
에디션 피드 상단에 유튜버 대표 명언·밈·유행어가 슬라이드로 순환 표시. "김근육이 말했다: ○○○" 형식. 팬은 클릭하면 해당 영상으로 이동. 유튜버 입장에서도 자기 콘텐츠가 앱 안에 살아있으니 소개할 명분이 생김.
구현 난이도 낮음 — DB 없이 하드코딩으로 시작 가능
3
에디션 전용 공유 카드 디자인
토론 공유 버튼을 누르면 유튜버 브랜딩이 입혀진 OG 이미지가 생성됨. 하단에 "월수 에디션에서 참여" 워터마크 포함. 카카오톡·X 공유 시 일반 카드와 시각적으로 달라 보임 → 팬들이 에디션 링크를 더 적극적으로 공유.
share.js의 thjMakeShareCard 확장으로 구현 가능
4
에디션 팬 배지
에디션 활성화 상태에서 댓글 작성 시 닉네임 옆에 "🥊 월수 팬" 배지 자동 부착. 배지는 해당 에디션에서만 보임. 팬들이 같은 팬임을 서로 알아보게 되는 커뮤니티 효과. 유튜버도 "내 팬들이 달고 다니는 배지"라는 소유감.
comments 테이블에 edition 컬럼 추가로 구현
5
에디션 전용 랭킹 보드
우측 랭킹 패널에 일반 랭킹 외에 "월수 에디션 TOP 5" 탭 추가. 에디션 추천 토론들의 참여 수 순위. 팬들이 랭킹을 올리기 위해 더 적극 참여하고, 유튜버도 자기 팬들이 만든 랭킹 결과를 영상 소재로 쓸 수 있음.
DB 태깅 없이도 picks 목록 기반 로컬 계산 가능
6
유튜버 픽 of the Week
에디션 추천 토론 중 매주 하나가 "이번 주 집중 토론"으로 에디션 배너에 크게 하이라이트. 유튜버가 영상 말미에 "이번 주 토론하자 주제는 ~~입니다"라고 언급하기 좋은 구조. 정기 콘텐츠 연동 포인트가 생김.
날짜 기반 자동 순환 — 서버 없이 구현 가능
7
에디션 ON 인트로 애니메이션
에디션 버튼을 눌러 활성화하는 순간, 유튜버 대표 색상으로 화면이 flash되며 "월수 에디션 ON" 텍스트가 0.6초 등장. 소소하지만 선택의 순간을 특별하게 만듦. 유튜버가 리뷰 영상에서 이 장면을 클립으로 쓸 수 있음.
CSS 애니메이션 단독 구현 — 추가 의존성 없음
8
에디션 한정 토론 개설 카테고리
토론 개설 화면에 "에디션 전용" 카테고리 추가. 에디션 활성화 중에만 보이고, 해당 카테고리로 개설된 토론은 에디션 피드 최상단에 노출. 팬들이 직접 에디션 테마에 맞는 토론을 만들어 채움 → 유튜버 없이도 콘텐츠가 자생적으로 증가.
create.html에 카테고리 옵션 추가 + DB 태그 컬럼 필요
